🎮 2048 Game
  The classic 2048 game , implemented in JavaScript.
  The goal of the game is to combine identical numbers to get a tile of 2048 (or even more 😏).

🚀 Getting started
To run the project locally, follow these steps:
  1. Clone the repository:
      git clone https://github.com/ExtymAndriy/2048game.git
      cd 2048game
  2. Install dependencies:
      npm install
  3. Launch the project:
      npm start

✨ Functions:
  🎯 Classic 2048 game mechanics
  ⬅️➡️⬆️⬇️ Key control (arrows)
  🔢 Automatic merging of identical tiles
  💾 Saving the best result
  🔄 Ability to start the game over

🧠 How to play
  Use the arrow keys to move the tiles.
  When two identical tiles touch, they merge.
  Reach the number 2048 to win
  The game ends when there are no more possible moves.
